# XploreNow

App de turismo desarrollada como Trabajo Práctico Obligatorio para la materia **Desarrollo de Aplicaciones I** (Prof. Brizuela, Horacio) en la Universidad Argentina de la Empresa.

XploreNow permite a los viajeros descubrir, reservar y guardar actividades turísticas en distintos destinos. La app combina autenticación segura, recomendaciones personalizadas según los gustos del usuario, gestión de favoritos con notificaciones de novedades, y un flujo completo de reserva con historial y reseñas.
